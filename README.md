## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/imranhossain2024/Portfolio-2.0.git
```

Go to the project directory:

```bash
cd Portfolio-2.0
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
